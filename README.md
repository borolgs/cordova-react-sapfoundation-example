# Cordova+React+SAP/Foundation
Example project

## Install
`git clone https://github.com/borolgs/cordova-react-sapfoundation-example.git`  
`npm i`

## Usage

#### dev
`cordova run browser -- --livereload`

#### build
`cordova prepare`  
`cordova build -- --webpackConfig webpack.config.js`  
`cordova run build ios`
